# sicd-rs

A rust SICD file interface

## Example

```rust
// Read sicd and print metadata
let file = std::path::Path::new("../example.nitf");
let sicd = sicd_rs::read_sicd(file).unwrap();
println!("{:?}", sicd.meta);
```
